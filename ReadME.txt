Este trabalho busca implementar um gerente de monitoramento em uma rede. Na segunda parte foi implementado um algoritmo de árvore para detectar anomalias, mais informações estão contidas nos relatórios.

Precisar estar com uma máquina gerente conectada com pelo menos um agente.

Para instalar as bibliotecas necessárias:
pip install --upgrade pip
pip install -r requirements.txt
